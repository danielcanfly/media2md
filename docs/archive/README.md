# Archive

This directory keeps release-engineering and acceptance-history artifacts that are useful to maintainers but are not part of the normal end-user installation or packaging surface.

Contents:

- `acceptance/`: historical strict-acceptance records
- `install-guides/`: historical versioned install runbooks
- `installers/`: historical versioned installer scripts
- `release/`: release manifests and handoff-oriented release notes

These files stay in the repository for traceability and regression coverage, but they are excluded from the user-facing source distribution package.
